Sparx - a game-oriented utility library
---------------------------------------
* Decodes PNG, JPEG2000 (JP2), DDS, TGA and BMP
* Loads MD5 models
* Calculates Bezier curves

This project is old and unmaintained. Feel free to copy it, modify and use for any tasks.
